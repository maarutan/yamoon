# usage

### Install

```bash
# Clone repository
mv ~/.config/yazi/ ~/.config/yazi-save
git clone https://github.com/maarutan/yazi-maaru.git ~/.config/yazi
cd ~/.config/yazi

# Install
./install.sh
```
